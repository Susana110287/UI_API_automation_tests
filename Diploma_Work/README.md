# Автоматизация UI‑ и API‑тестов
## Цель проекта: автоматизировать UI‑ и API‑тесты на основе финальной работы по ручному тестированию (или альтернативного проекта).

## Описание задачи
Проект реализует автоматизацию:

- не менее 5 UI‑тестов (на основе функционального чек‑листа из финальной работы по ручному тестированию);
- не менее 5 API‑тестов (на основе коллекции и кейсов из финальной работы).

Поддерживается запуск тестов в трёх режимах:

- только UI‑тесты;
- только API‑тесты;
- все тесты.

## Выбор проекта
Использование финальной работы по ручному тестированию.
